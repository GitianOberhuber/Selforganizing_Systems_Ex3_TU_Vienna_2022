This is our project for the lecture Selforganizing Systems at TU Vienna, Exercise 3. We extend the MiniSom library (https://github.com/JustGlowing/minisom) by adding options to train mnemonic SOMs. We then use SOMToolb (https://github.com/smnishko/SOMToolbox/) for visualizing our results. 

[1] Mayer, Rudolf, Dieter Merkl, and Andreas Rauber. Mnemonic SOMs: Recognizable shapes for self-organizing maps, 2005.
